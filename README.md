# Netflix-Customer-Churn-Analysis# 📊 Netflix Customer Churn Analysis

## 📝 Project Overview

This project focuses on identifying the critical drivers of customer churn within a subscription-based service. By leveraging a dataset of 5,000 records, I developed an end-to-end machine learning pipeline to preprocess data, engineer relevant features, and predict subscriber retention. The objective was to translate complex model outputs into actionable insights that support strategic business decision-making.

## 🛠️ Technical Stack

* **Languages:** Python
* **Libraries:** Pandas, Scikit-Learn
* **Methodology:** Data Cleaning, Feature Engineering, Logistic Regression

## 🚀 Key Accomplishments

* **Data Processing:** Developed a robust machine learning pipeline to clean, process, and engineer key features from a 5,000-record dataset to effectively isolate factors influencing customer churn.
* **Model Implementation:** Implemented a Logistic Regression model that achieved **89% predictive accuracy** for customer retention.
* **Business Impact:** Successfully translated technical model outputs into actionable business intelligence to guide subscriber management and retention strategies.

## 📂 Repository Contents

* `data/`: Contains the raw and processed datasets used for the analysis.
* `notebooks/`: Jupyter Notebook detailing the entire workflow, from data cleaning and feature engineering to model training and evaluation.
* `scripts/`: Python scripts used for pipeline automation.
